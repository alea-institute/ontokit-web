Each prompt module follows the folio-enrich `concept_identification.py` pattern (D-02, GEN-07): a `SYSTEM` constant + a `build_messages(context: dict, batch_size: int) -> list[dict[str, str]]` function.

**All prompts share this pattern:**
- System message: ontology expert role + JSON-only output instruction
- User message: structured ontology context from assembler + specific request + JSON output schema
- Output schema always includes `"suggestions": [{"label": str, "definition": str, "confidence": 0.0-1.0, ...}]`

**children.py** — asks for `batch_size` child classes. Context includes current class label, parents, existing siblings, existing children. JSON output: `{"suggestions": [{"label": str, "definition": str, "confidence": float}]}`

**siblings.py** — asks for `batch_size` sibling classes under the same parent. Context includes current class, parent class, existing siblings. JSON output: same as children.

**annotations.py** — asks for `batch_size` annotation suggestions (altLabel, skos:example, rdfs:comment translations, skos:note). Context includes current class, existing annotations, existing labels. JSON output: `{"suggestions": [{"property_iri": str, "value": str, "lang": str, "confidence": float}]}` where property_iri is the annotation property IRI.

**parents.py** — asks for `batch_size` additional parent classes. Context includes current class, current parents, siblings. JSON output: `{"suggestions": [{"label": str, "iri": str (if existing class), "definition": str, "confidence": float}]}`

**edges.py** — asks for `batch_size` seeAlso/isDefinedBy resource suggestions. Does NOT ask for URIs directly (per research open question 3). Instead asks for resource descriptions and search queries. JSON output: `{"suggestions": [{"property_iri": str, "target_description": str, "search_query": str, "confidence": float}]}` where property_iri is "rdfs:seeAlso" or "rdfs:isDefinedBy".

**Default batch sizes** (used in system message hints, actual batch_size comes from caller):
- children: 5, siblings: 5, annotations: 5, parents: 3, edges: 3
