   - Else: Query IndexedEntity for this project/branch, GROUP BY extracted namespace prefix (text before last `#` or `/`), return most common
   - Fallback: `f"http://example.org/ontology/{project_id}#"`

**ValidationService class:**

```python
class ValidationService:
    def __init__(self, db: AsyncSession):
        self._db = db
        self._index = OntologyIndexService(db)

    async def validate_entity(
        self,
        project_id: UUID,
        branch: str,
        entity: dict,          # {"label": ..., "parent_iris": [...], "iri": ..., "labels": [...]}
        project_namespace: str, # from detect_project_namespace
    ) -> list[ValidationError]:
```

**Validation rules (each returns list[ValidationError]):**

- `_check_parent_required(entity)` — VALID-01: `entity["parent_iris"]` must be non-empty list. Error: `ValidationError(field="parent_iris", code="VALID-01", message="At least one parent class is required. New classes cannot be free-floating entities.")`

- `_check_english_label(entity)` — VALID-02: `entity["labels"]` must contain at least one entry where `lang` is `"en"` or `""` (empty string = language-untagged treated as English). Error: `ValidationError(field="labels", code="VALID-02", message="An English rdfs:label is required. Add a label with language tag 'en'.")`

- `async _check_cycles(entity, project_id, branch)` — VALID-03: For each IRI in `entity["parent_iris"]`, query `self._index.get_ancestor_path(project_id, branch, parent_iri)`. If `entity["iri"]` appears in any ancestor path, it would create a cycle. This is the LIGHTWEIGHT SQL-based check (not full OWL parse — per Pitfall 2 in RESEARCH.md). Error: `ValidationError(field="parent_iris", code="VALID-03", message=f"Assigning parent '{parent_iri}' would create a cycle: {entity['iri']} is already an ancestor of {parent_iri}.")`

- `_check_namespace(entity, project_namespace)` — VALID-04: Extract namespace from `entity["iri"]` (text up to and including last `#` or `/`). If it doesn't match `project_namespace`, block. Skip check if `entity["iri"]` is empty/None (IRI not yet minted). Error: `ValidationError(field="iri", code="VALID-04", message=f"IRI namespace '{extracted}' is not owned by this project. Expected namespace: '{project_namespace}'.")`

- `validate_entity()` orchestrates all four checks, returns combined error list (VALID-05: structured errors with field+code+message).
