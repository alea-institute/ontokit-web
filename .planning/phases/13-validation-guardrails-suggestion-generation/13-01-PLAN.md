**`mint_iri(namespace: str) -> str`** — module-level function per D-11:
- Generate UUID v4 via `uuid.uuid4()`

- Ensure it ends with `#` or `/` (append `#` if not)
- Fallback: `f"http://ontology.example.org/{project.id}#"`

**`class ValidationService`**:
- `__init__(self, db: AsyncSession)` — stores db, creates `OntologyIndexService(db)`
- `async def validate_entity(self, project_id: UUID, branch: str, entity: dict, project_namespace: str) -> list[EntityValidationError]` — runs all rules, returns aggregated list:
  1. `_check_parent_required(entity)` — VALID-01: checks `entity.get("parent_iris", [])` is non-empty. Error message: `"At least one parent class is required. New entities cannot be free-floating."`
  2. `_check_english_label(entity)` — VALID-02: checks that `entity.get("labels", [])` contains at least one item with `lang in ("en", "")`. Also checks `entity.get("label", "")` as fallback for simple dict format. Error message: `"An English label (rdfs:label with lang='en') is required."`
  3. `async _check_cycles(entity, project_id, branch)` — VALID-03: for each IRI in `entity.get("parent_iris", [])`, query ancestors of that parent via SQL recursive CTE (same pattern as `get_ancestor_path`). If `entity.get("iri")` appears in the ancestor set, it is a cycle. Error message: `f"Assigning parent '{parent_iri}' would create a cycle: {entity_iri} is already an ancestor of {parent_iri}."`
  4. `_check_namespace(entity, project_namespace)` — VALID-04: extracts namespace prefix from `entity.get("iri", "")` by splitting at last `#` or `/`. If entity has no IRI (None or empty), skip (IRI not yet minted). If prefix does not match `project_namespace`, error. Error message: `f"IRI namespace '{prefix}' is not owned by this project. Expected: '{project_namespace}'."`

For the cycle detection SQL, use a recursive CTE on `indexed_hierarchy` table:
```sql
WITH RECURSIVE ancestors AS (
    SELECT parent_iri FROM indexed_hierarchy
    WHERE project_id = :pid AND branch = :branch
      AND child_iri = :start AND parent_iri != :owl_thing
    UNION ALL
    SELECT h.parent_iri FROM indexed_hierarchy h
    JOIN ancestors a ON h.child_iri = a.parent_iri
    WHERE h.project_id = :pid AND h.branch = :branch
      AND h.parent_iri != :owl_thing AND a.depth < 100
)
SELECT 1 FROM ancestors WHERE parent_iri = :target LIMIT 1
```
Use `sqlalchemy.text()` for the raw SQL. The `owl_thing` constant is `"http://www.w3.org/2002/07/owl#Thing"`.
