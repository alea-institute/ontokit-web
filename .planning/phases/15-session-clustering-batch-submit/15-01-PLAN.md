}

**Store state interface:**

```typescript
interface ShardPreviewState {
  prGroups: Record<string, PRGroupDefinition>;
  shards: Record<string, ShardDefinition>;
  prGroupOrder: string[];        // ordered PR group IDs for stable rendering
  expandedPrIds: Set<string>;    // which PR groups are expanded
  expandedShardIds: Set<string>; // which shards are expanded

  // Hydration
  setFromClusterResponse: (response: ClusterResponse) => void;

  // Mutations (all atomic — single set() call)
  moveEntity: (entityIri: string, fromShardId: string, toShardId: string) => void;
  mergeShards: (sourceId: string, targetId: string) => void;
  splitShard: (shardId: string, entityIris: string[], newLabel: string) => void;
  moveShard: (shardId: string, fromPrId: string, toPrId: string) => void;

  // Expand/collapse
  togglePrExpanded: (prId: string) => void;
  toggleShardExpanded: (shardId: string) => void;

  // Reset
  clear: () => void;

  // Derived
  getSummary: () => { totalSuggestions: number; totalShards: number; totalPrs: number };
}
```

**Import the ClusterResponse type** from `lib/api/suggestions.ts` (created in Task 2, but the type is defined here first so both files share it).

**Mutation implementation requirements:**
- `moveEntity`: Single `set()` call. Remove from `source.entityIris`, add to `target.entityIris`. Recompute `suggestionCount` on both affected PR groups.
- `mergeShards`: Single `set()` call. Combine `entityIris`. Delete source shard from `shards`. Remove source ID from its PR group's `shardIds`. Recompute `suggestionCount`.
- `splitShard`: Single `set()` call. Generate new shard ID via `crypto.randomUUID()`. Create new `ShardDefinition` with the provided `entityIris` and `newLabel`. Remove those IRIs from the original shard. Add new shard ID to the same PR group. Recompute `suggestionCount`.
- `moveShard`: Single `set()` call. Remove shard ID from source PR group's `shardIds`, add to target. Recompute both `suggestionCount` values.

**Expand/collapse defaults:**
- `setFromClusterResponse`: If total shards > 3, all PR groups start collapsed. Otherwise, all expanded. All individual shards start collapsed regardless.
