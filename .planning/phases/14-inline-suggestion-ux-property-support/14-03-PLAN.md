**Suggestion hook instances** â€” create one `useSuggestions` call per section type. Use 6 instances for the 6 sections that get suggestion support:
- `childrenSuggestions` = `useSuggestions({ ..., suggestionType: "children", onAccepted: handleAcceptChildSuggestion })`
- `siblingsSuggestions` = `useSuggestions({ ..., suggestionType: "siblings", onAccepted: handleAcceptChildSuggestion })`
- `annotationsSuggestions` = `useSuggestions({ ..., suggestionType: "annotations", onAccepted: handleAcceptAnnotationSuggestion })`
- `parentsSuggestions` = `useSuggestions({ ..., suggestionType: "parents", onAccepted: handleAcceptParentSuggestion })`
- `edgesSuggestions` = `useSuggestions({ ..., suggestionType: "edges", onAccepted: handleAcceptEdgeSuggestion })`

Common opts passed to all: `{ projectId, entityIri: classIri, branch: branch ?? "main", canUseLLM: canUseLLM ?? false, accessToken, byoKey }`

**Accept flow routing** (per Pitfall 1 in RESEARCH.md â€” CRITICAL):

1. `handleAcceptChildSuggestion(suggestion, editedValue)`:
   - Routes through `onAddSuggestedChild?.(suggestion.iri, editedValue ?? suggestion.label, classIri!)` (per D-07 â€” accepted classes appear in tree immediately)
   - Does NOT merge into edit state â€” child suggestions create NEW entities

2. `handleAcceptAnnotationSuggestion(suggestion, editedValue)`:
   - Merges into `editAnnotations` state: add a new annotation entry with `property_iri: suggestion.property_iri`, `values: [{ value: editedValue ?? suggestion.value ?? suggestion.label, lang: suggestion.lang ?? "en" }]`
   - Calls `triggerSave()` to write to draft store

3. `handleAcceptParentSuggestion(suggestion, editedValue)`:
   - Adds `suggestion.iri` to `editParentIris` via `addParent(suggestion.iri, editedValue ?? suggestion.label)`
   - Calls `triggerSave()`

4. `handleAcceptEdgeSuggestion(suggestion, editedValue)`:
   - Adds to `editRelationships` state: new entry with `property_iri: suggestion.relationship_type`, `target: suggestion.target_iri`, `label: editedValue ?? suggestion.label`
   - Calls `triggerSave()`
